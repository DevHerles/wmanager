# wmanager

Window manager dotfiles

If Netbeans (java) shows empty window in tiling window manager (awesome bspwm)

```bash
sudo apt-get install suckless-tools
```

Then

```bash
wmname LG3D
```

Or

```bash
wmname compiz
```
